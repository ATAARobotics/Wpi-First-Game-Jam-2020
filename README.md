# Wpi-First-Game-Jam-2020

Designed for the 2020 WPI and FIRST game jam, this is a collaborative platformer parkour, where you control two players, who together must get to the very end of the level.

# Controls:

WASD: Movement
Space: Jump
CTRL: Crouch
Mouse: Camera
E: Swap Players
-: Restart Level
